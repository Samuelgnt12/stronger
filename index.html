POWERARM

Program Pembesaran Lengan & Penguatan Tangan Khusus Panco

LEVEL PEMULA: PERBAIKI LENGAN KURUS

1. Dumbell Curl Dasar

Latihan dasar untuk membangun massa otot biceps.

  • 3 set x 10-12 repetisi
  • Istirahat 60 detik antar set
  • Gunakan beban yang bisa dikontrol

2. Grip Squeeze

Menguatkan cengkeraman tangan dengan hand gripper.

  • 5 set x 5 repetisi (tahan 5 detik)
  • Istirahat 45 detik antar set

LEVEL MENENGAH: 5 5 5 PROGRAM

1. 5x5 Weighted Chin-up

Latihan compound untuk kekuatan lengan dan punggung.

  • 5 set x 5 repetisi
  • Tambahkan beban jika perlu
  • Istirahat 90 detik antar set

2. 5x5 Wrist Roller

Membangun kekuatan lengan bawah dan cengkeraman.

  • 5 set x 5 gulungan penuh
  • Gunakan beban progresif

LEVEL LANJUTAN: PEMBESARAN LENGAN SUPER

1. One-Arm Deadhang dengan Beban

Latihan ekstrim untuk kekuatan cengkeraman dan lengan.

  • 5 set x 5-10 detik tahan
  • Gunakan sabuk pemberat

2. Plate Pinch Challenge

Latihan khusus untuk kekuatan jari dan cengkeraman.

  • 5 set x 5-10 detik tahan
  • Gunakan dua plate barbel yang disatukan

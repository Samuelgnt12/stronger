Power Grip Training

Durasi: 45 menit
Intensitas: Tinggi
Alat: Dumbell, Gripper

Instruksi Latihan

  1. Pemanasan: Lakukan wrist circles selama 2 menit
  2. Heavy Gripper: 4 set x 8 repetisi (tahan 3 detik)
  3. Plate Pinch Hold: 3 set x 15 detik
  4. Towel Hang: 3 set x maksimal durasi
  5. Dumbell Wrist Curl: 3 set x 12 repetisi

Tips Ahli

  • Fokus pada kontraksi penuh setiap repetisi
  • Jaga nafas tetap stabil
  • Rekam waktu tahan untuk memantau progres
  • Istirahat 90 detik antar set
  • Hentikan jika merasakan sakit berlebihan
